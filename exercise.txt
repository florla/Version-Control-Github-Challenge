Explain what the terminal is in your own words.
The terminal is a system with no graphical user interface used to send commands to a computer and help automate processes.

What is the major difference between Git and GitHub?
Git is a version control system to help you keep track of your code while GitHub is a website designed to host Git repositories.

What are the steps to fork a GitHub repository?
step 1: navigate to the repository you want to fork
step 2: press the fork button at the top right of the repo
step 3: choose the owner, change the repository name, and add a description, and choose if you would like  to copy the main branch only (most of this is optional) 
step 4: click create fork
